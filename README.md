# AI Lead Scoring Dashboard

An AI-powered Lead Scoring Dashboard built with Streamlit and Groq.

## Features

- AI Lead Scoring
- Priority Classification
- AI Reasoning
- Recommended Next Action
- Interactive Dashboard
- Charts
- Search
- Filters

---

## Installation

Clone repository

```bash
git clone <repo-url>

cd lead-scoring-dashboard
```

Install packages

```bash
pip install -r requirements.txt
```

Create .env

```env
GROQ_API_KEY=your_key
```

Run

```bash
streamlit run app.py
```
